## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Sujal-cloud/sujal-portfolio.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```
